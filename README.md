# UK House Price Analysis

This project explores UK housing price data using Python.  
The goal is to clean the dataset, verify sales volume consistency,
and analyse regional housing trends.

## Dataset

Source: UK House Price Index

The dataset includes information such as:

- Region
- Sales volume
- Mean house price
- New and old property sales
- Detached house prices
- Flat prices

## Tools Used

- Python
- pandas
- numpy
- seaborn
- matplotlib
- Jupyter Notebook

## Analysis Performed

The notebook includes:

- Data cleaning and preparation
- Verification of sales volume consistency
- Regional sales aggregation
- House price analysis for Greater Manchester
- Median detached house price analysis
- Construction of a House Price Index (HPI)
- Identification of seasonal price patterns
